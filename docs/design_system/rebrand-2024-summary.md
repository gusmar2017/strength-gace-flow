# Rebrand Summary - December 2024

## Overview
Updated brand color palette to align with new logo design featuring terracotta, dusty blue, and clay neutral tones.

## Color Palette Changes

### Before (Sage & Clay Era)
- Primary: Muted Sage `#8FAEA3`
- Secondary: Warm Clay `#C7A89A`
- Accent: Same as secondary

### After (Terracotta & Blue Era)
- **Primary: Deep Terracotta `#B35C44`** - Warm, grounding, earthy
- **Secondary: Dusty Blue `#6F8F9B`** - Calm, balanced, flowing
- **Tertiary: Soft Clay Neutral `#D8B8A6`** - Gentle, nurturing, soft

## Visual Impact

### Brand Personality Shift
**Before:** Sage green wellness (botanical, garden-inspired)
**After:** Earthy terracotta + flowing blue (grounded strength + fluid movement)

### Color Psychology
- **Terracotta:** Grounding, warmth, earth connection, strength
- **Dusty Blue:** Flow, calm, water element, grace
- **Clay Neutral:** Softness, nurturing, balance

Aligns perfectly with **"Strength, Grace & Flow"**:
- Strength = Terracotta
- Grace = Clay Neutral
- Flow = Dusty Blue

## Cycle Phase Colors

| Phase | Color | Hex | Feeling |
|-------|-------|-----|---------|
| **Menstrual** | Deep Terracotta | `#B35C44` | Grounding, restful |
| **Follicular** | Dusty Blue | `#6F8F9B` | Calm renewal, rising energy |
| **Ovulatory** | Soft Clay | `#D8B8A6` | Gentle vitality, warmth |
| **Luteal** | Blue-Clay Blend | `#9B9A9A` | Balanced transition |

## Logo Integration

New logo features:
- Woman in contemplative/flowing pose
- Moon phases (cyclical nature)
- Botanical elements (natural connection)
- Wave pattern (flow element)
- Uses all three core colors harmoniously

## Implementation

**Files Updated:**
- `docs/design_system/design-system.md` - Core palette documentation
- `StrengthGraceFlow/Core/Theme/SGFTheme.swift` - App color definitions

**Visual Changes:**
- All buttons: Sage → Terracotta
- Tab bar active: Sage → Terracotta
- Secondary elements: Now use Dusty Blue
- Phase indicators: Updated to new palette

## Brand Alignment

This rebrand strengthens the connection between:
1. **Visual identity** (logo colors) ↔ **App colors**
2. **Brand name** (Strength, Grace & Flow) ↔ **Color meaning**
3. **Cyclical wellness** (moon phases) ↔ **Phase colors**

The palette is more distinctive, memorable, and true to the core brand essence.
